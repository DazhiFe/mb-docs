# 开关

## 基础用法

```html
<label class="label-switch">
  <input class="switch-hook" type="checkbox" checked />
  <i class="icon-switch"></i>
</label>
```

## 禁用状态

通过给`class`属性添加`disabled`值可以禁用开关。

```html
<label class="label-switch disabled">
  <input class="switch-hook" type="checkbox" checked />
  <i class="icon-switch"></i>
</label>
```
